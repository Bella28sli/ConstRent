Горячие клавиши

  • Alt + H — главная
  • Alt + E — оборудование
  • Alt + R — создать аренду
  • Alt + L — все аренды
  • Alt + P — оплаты аренды
  • Alt + C — клиенты
  • Alt + K — история аренды клиента
  • Alt + D — долги по оплате
  • Alt + B — долги по оборудованию
  • Alt + M — ТО (календарь)
  • Alt + U — требуется ТО
  • Alt + O — CSV импорт/экспорт
  • Alt + S — настройки профиля
  • Alt + T — переключить тему
  • Alt + F — перейти к поиску
